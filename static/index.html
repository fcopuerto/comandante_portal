How to get the driver files
  1. Go to support.brother.com, search your printer model and select Windows.
  2. Download the Full Driver & Software Package for each OS version you need.
  3. Run the EXE — when the installer asks to proceed, cancel it. The files are already extracted.
  4. Open C:\Users\<you>\AppData\Local\Temp\Brother\ and find the folder with .inf files.
  5. Select all files in that folder (INF + CAB + CAT) and upload them in the slot below.
  6. Fill in the Windows driver name exactly as it appears inside the INF file (look for DriverDesc= or DriverPackageDisplayName=).

How to import your WireGuard config
  1. Download and install WireGuard from the Apps section above or from wireguard.com/install
  2. Download your .conf file using the button in the table.
  3. Open WireGuard → click Import tunnel(s) from file… → select the .conf file.
  4. Click Activate.

PowerShell (run as Administrator):

wireguard.exe /installtunnel "C:\Users\YourName\Downloads\cobaltax.conf"
  1. Install WireGuard from the Mac App Store, or via Homebrew: brew install wireguard-tools
  2. Download your .conf file.
  3. Open WireGuard → File → Import Tunnel(s) from File… → select the file.
  4. Click Activate.

Terminal (wg-quick):

sudo wg-quick up ~/Downloads/cobaltax.conf
  1. Install WireGuard: sudo apt install wireguard (Debian/Ubuntu) or sudo dnf install wireguard-tools (Fedora/RHEL)
  2. Download your .conf file.
  3. Copy to /etc/wireguard/: sudo cp cobaltax.conf /etc/wireguard/cobaltax.conf
  4. Start the tunnel: sudo wg-quick up cobaltax
  5. Enable on boot: sudo systemctl enable wg-quick@cobaltax
  1. Install the WireGuard app from the App Store (iOS) or Google Play (Android).
  2. Download the .conf file to your device (email it to yourself or use AirDrop).
  3. Open WireGuard → tap + → Create from file or archive → select the .conf file.
  4. Toggle the tunnel on.

Tip: On iPhone you can AirDrop the .conf file directly from your Mac.
